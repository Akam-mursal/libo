# Trading Signal Dashboard

A real-time trading signal dashboard that displays market data and trading signals.

## Features

- Real-time market data visualization
- Trading signal generation and tracking
- Performance statistics (wins, losses, win rate)
- Export functionality for signal data
- Responsive design for all devices

## Setup

1. Clone this repository:
   ```
   git clone https://github.com/Akam-mursal/df.git
   ```

2. Open `index.html` in your web browser

## Usage

1. Click "Start Monitoring" to begin receiving signals
2. View real-time market data and signals
3. Use the export button to save signal data as CSV

## File Structure

- `index.html` - Main HTML file
- `css/style.css` - Styles for the dashboard
- `js/app.js` - JavaScript functionality
